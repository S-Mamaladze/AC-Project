📌 Project Overview
The Creed Chronicles is a comprehensive data analytics project that explores the evolution of the Assassin’s Creed franchise from its inception in 2007 to the upcoming Shadows in 2025.
The project solves a "Business Intelligence" problem: How has the shift from linear action-adventure to massive RPGs affected the franchise's financial ROI, content density, and player engagement?

🚀 Key Features
Normalized SQL Database: A robust schema of 15+ tables tracking everything from financial budgets to Piece of Eden lore.
Advanced Analytics: Custom KPIs like "Content Bloat Factor," "Mission Density," and "Revenue Efficiency."
Interactive UI/UX: A high-end Power BI dashboard designed with a custom dark/gold "Animus" theme.
Drill-Through Capabilities: Seamless navigation from a high-level "Global Ledger" to a granular "Game Archive" and "Individual Game Dossiers."

🛠️ Tech Stack
Database: Microsoft SQL Server (T-SQL)
Data Modeling: Relational Star Schema (3rd Normal Form)
Analytics & Visualization: Power BI Desktop
DAX (Data Analysis Expressions): Used for complex time-series and metric calculations.
Documentation: Markdown

📐 Data Architecture
The database is structured to ensure zero redundancy and high performance. Key tables include:
Fact_Games: Core game statistics and playtime hours.
Dim_Financials: Budget, Revenue, and Profitability metrics.
Dim_Map_Stats: Map size (km^2), continents, and regional counts.
Dim_Gameplay: Skill systems, combat archetypes, and naval mechanics.
Dim_Lore: Historical figures, landmarks, and artifacts.

📊 Business Insights Developed
The RPG Pivot: Identified that while development budgets grew by 800%, the move to the RPG era resulted in the franchise's first $1 Billion title (Valhalla).
The Density Paradox: Visualized how map sizes increased by over 1,000%, but "Mission Density" dropped significantly, indicating a shift in world-building philosophy.
The Bloat Gap: Analyzed the increasing gap between Main Story length and 100% Completion time, highlighting the "Value for Money" strategy in newer titles.
